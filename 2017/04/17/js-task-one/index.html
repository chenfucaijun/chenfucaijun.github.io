DOM基础操作

DOM = Document Object Model，文档对象模型。DOM定义了表示和修改文档所需的对象、这些对象的行为和属性以及这些对象之间的关系。可以把DOM抽象成一个树形表示它的层级结构。如下图:

原生Js中如何获取DOM中的节点元素?

  • 直接获取
    • getElementById(): 方法可返回对拥有指定 ID 的第一个对象的引用。注意:在一个页面中,DOM节点元素的ID是唯一的!通过这种获取方式能
    • getElementsByTagName():方法可返回带有指定标签名的对象的集合。标签可以重复,所以通过标签取某个特定元素不是明智之举
    • getElementByCLassName():方法可返回带有指定类名的对象的集合。 class可以不唯一,同一个元素可以有多个class名。一般class用来统一设置样式。
    • getElementsByName() 方法可返回带有指定名称的对象的集合。name不唯一,可用在多个dom节点上
  • 间接获取
    • element.childNodes:返回元素子节点的 NodeList。从树中查看就是返回该树节点的子节点列表
    • element.firstChild:返回元素的首个子节点。
    • element.lastChild:返回元素的最后一个子元素。
    • element.parentNode():返回元素的父节点。
    • element.previousSibling:返回位于相同节点树层级的前一个元素(返回上一个兄弟元素)。

      这些访问节点的方式在jQuery中都有所封装,获得更便捷的操作。

通过JS操作节点

  • 对JS中的DOM节点进行的操作主要有
    • 获取或者设置节点元素中的内容
      • innerHTML :属性设置或返回该标签内的HTML。也就是从对象的起始位置到终止位置的全部内容,包括Html标签。
      • outerHTML：属性设置或返回该标签及标签内的HTML。也就是从对象的该标签起始到终止位置的全部内容,包括Html标签。
        如果要输出不含HTML标签的内容，可以使用innerHTML取得包含HTML标签的内容后，再用正则表达式去除HTML标签，
      • inneText：从起始位置到终止位置的内容，但它去除html标签。（只能在IE和chrome下使用）
      • outerText 设置(包括标签)或获取(不包括标签)对象的文本
      • value：属性可设置或返回密码域的默认值。获取文本框的值

为DOM元素添加事件

  • addEventListener() 方法用于向指定元素添加事件句柄。

    • 为 <button> 元素添加点击事件。 当用户点击按钮时，在 id=”demo” 的

      元素上输出 “Hello World” :

  • 语法
    element.addEventListener(event, function, useCapture)

    • event:必须。字符串，指定事件名。HTML DOM 事件
    • function: 必须。指定要事件触发时执行的函数。
    • 可选。布尔值，指定事件是否在捕获或冒泡阶段执行。
      • true - 事件句柄在捕获阶段执行
      • false- false- 默认。事件句柄在冒泡阶段执行
